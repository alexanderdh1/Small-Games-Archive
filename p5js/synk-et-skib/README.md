# Synk Et Skib (p5.js)
A Battleships-style game developed as a mandatory school project.
The project was not fully completed, but core structure and logic were implemented.

## Project status
The game allows grid creation and partial ship placement.
Several planned features were not implemented, including:
- Placement of all ships
- Computer opponent
- Collision detection
- Turn-based gameplay

## Included material
- Source code (p5.js)
- Project synopsis describing intended design, structure, and improvements

## Context
This project is included as a learning exercise and does not represent
a finished or polished product.
