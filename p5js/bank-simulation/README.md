# Bank Simulation (p5.js)

A small bank account simulation created as a school project using p5.js.

The program simulates a simple bank account with balance and cash handling,
including deposits, withdrawals, monthly income, and interest.

## Features
- Deposit and withdraw money
- Monthly salary
- Interest applied to positive balance
- Simple graphical user interface
- Input-based transactions

## Implementation
The project uses object-oriented programming with separate classes for
bank account logic and the graphical user interface.

## Context
This project is included as a learning exercise and does not represent
a production-ready financial application.
