# My First Game (PICO-8)
A small game created in 2021 using the PICO-8 fantasy console.

## File format
The game is stored as a `.p8.png` cartridge file.  
The PNG contains the full game code and visuals:
- Lua source code
- Sprites and map data
- Sound and music

## How to open and run
1. Open PICO-8
2. Load the cartridge
3. Run the game
